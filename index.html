NECTIN-4 Group Definitions
Group 1: NECTIN-4/CEP1 ratio ≥ 2.0 AND average NECTIN-4 copy number ≥ 6.0 → Positive Group 2: NECTIN-4/CEP1 ratio ≥ 2.0 AND average NECTIN-4 copy number < 6.0 → Positive Group 3: NECTIN-4/CEP1 ratio < 2.0 AND average NECTIN-4 copy number ≥ 6.0 → Positive Group 4: Borderline range: ratio 1.8–2.2 or average NECTIN-4 copy number 5.4–6.6 → Count 20–50 additional cells Group 5: ratio < 2.0 AND average NECTIN-4 copy number < 6.0, without Group 4 borderline features → Negative
Heterogeneity Definitions
Low heterogeneity: CV 0.25–0.40 Moderate heterogeneity: CV > 0.40–0.60 High heterogeneity: CV > 0.60 Below low heterogeneity range: CV < 0.25
Chromosome Gain Definitions
No chromosome gain: Average CEP1 < 2.25 Low-level chromosome gain: Average CEP1 2.25 to < 3.0 Moderate chromosome gain: Average CEP1 3.0 to < 4.0 Marked chromosome gain: Average CEP1 ≥ 4.0 Chromosome gain is not the same as focal amplification. A high NECTIN-4 count with proportionally increased CEP1 counts may reflect chromosome 1 gain rather than focal amplification.
Interpretive Note
This calculator uses a HER2-analogous research framework for NECTIN-4. Score invasive neoplastic cells only. Exclude in situ elements, benign cells, necrotic areas, and overlapping nuclei. Only score nuclei showing at least one NECTIN-4 signal and at least one CEP1 signal. If the result falls into Group 4, count 20–50 additional tumor cells before final classification. Polysomy / chromosome gain should be distinguished from focal amplification.
